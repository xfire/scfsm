scala finite state machine
==========================

a simple scala based fsm implementation.


building and running
--------------------

### requisites

* installed [jdk](http://java.sun.com/) 1.5 or higher
* [sbt](http://code.google.com/p/simple-build-tool/) >= 0.7.1

### compile

    > sbt update
    > sbt compile

### run

    > sbt test
